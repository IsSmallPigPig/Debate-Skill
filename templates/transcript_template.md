# Transcript Template · 辩论实录模板

> 使用方式：复制本模板 → 替换 `{...}` 占位符 → 输出为 `辩论实录-{topic_slug}.md`。
>
> **可直接复制到 Word 打印**。所有文本必须符合 `rules/script_format.md` 的规范：
> 不带动作括号、不写小说旁白、使用真实辩场套语。

## 模板自检清单

打印本实录前，编排器必须自检（参考 `rules/script_format.md` §7.3 + §9）：

- [ ] 通篇**无**（动作/表情/语气）括号
- [ ] 通篇**无**小说式旁白（"XX 站起" "XX 苦笑"等）
- [ ] 通篇**无**环境描写（"现场气氛紧张"等）
- [ ] 通篇**无** AI 元说明（"作为 AI/作为 X 辩手"等）
- [ ] 所有 `{占位符}` 已替换
- [ ] 所有段落之间有空行
- [ ] 中文标点统一全角
- [ ] 引用资料带 URL
- [ ] 表格列对齐
- [ ] 标题层级不跳级（`#` → `##` → `###`）
- [ ] 每轮的 `> 时长 X'X" · 计时方式：—` 副标题齐全

任一项不达标 → 必须修订后才能交付。
