# 立论与战场

## 一、立论"闭环论证链"五元素

> **定义 → 标准 → 论点 → 论据 → 价值**
> —— 任何缺一环，对方都能从那一环把你打穿。

| 元素 | 作用 | 一辩稿对应位置 |
|---|---|---|
| **定义** | 界定关键词，框定讨论范围 | 开宗明义段 |
| **标准（判准）** | 给评委一把尺子，让"赢"可被判定 | 判准段 |
| **论点** | 在标准下我方为什么对 | 论点一/二/三 |
| **论据** | 支撑论点的数据 / 案例 / 学理 | 每个论点内 |
| **价值** | 这场辩论守护的是什么 | 结尾升华段 |

---

## 二、定义战

### 1. 定义的三种打法

| 打法 | 怎么做 | 示例 |
|---|---|---|
| **本质定义** | 抓事物最本质的特征 | "线下书店 = 实体 + 卖书"——不能否认这两点 |
| **延展定义** | 把概念在时代语境里往后推 | "婚姻"今天的含义 ≠ 古代的含义 |
| **价值定义** | 直接绕过事实，从价值层切入 | "我方不讨论书店是什么，只讨论书店在今天还重要不重要" |

### 2. 定义战的红线（养喵园 / 知乎）

- ❌ **不要为了打赢而定义**：定义不能违反基本常识，否则评委不接受；
- ❌ **不要陷入定义循环**："好不好" → "好" → "符合判准" → 判准还是"好" —— 听君一席话如听一席话；
- ✅ **本质特征不可否认**：超市卖书但超市不是书店——抓的是"主营业务"这个本质；
- ✅ **常识范围内的解释优先**：太奇异的定义即使在逻辑上自洽也会被评委直接判输。

### 3. 定义战的反击

如果对方的定义不利于你：

| 反击方式 | 模板 |
|---|---|
| 找反例 | "按您方定义，X 就不算 Y 了，可常识上 X 显然是 Y" |
| 比来源 | "您方定义出自哪里？我方援引《现代汉语词典》..." |
| 比覆盖 | "您方定义只能解释 30% 的情况，我方定义能覆盖 100%" |
| 比价值 | "采用您方定义后，这场辩论将失去讨论价值" |

## 三、判准（标准）

### 1. 判准 = 定义的一部分（养喵园核心观点）

> "判断标准本身就是定义。" —— 知乎养喵园

例：政策辩"该不该推行 X" —— 通说判准是"需根解损"（需要 / 根属 / 解决 / 损益）。
"需根解损"本身就是"该"的定义。所以不要在一辩稿里**又**写一段"判准是 X"——
直接把判准融进定义段即可。

### 2. 三种典型判准

| 类型 | 适用 | 示例 |
|---|---|---|
| **目的论判准** | 大多数辩题 | "哪种方式更能达成 X 这个目的" |
| **利弊比较判准** | 利弊型辩题 | "比较 A、B 哪个利大于弊" |
| **价值优先判准** | 价值型辩题 | "在 X 价值与 Y 价值冲突时优先选哪个" |

### 3. 判准的设定艺术

> **关键：让你的判准在场上看似中立，实则倾向你方。** —— 知乎匿名

例："正义战胜邪恶是否是儿童文学的必须结局"
- 烂判准："是否有利于儿童的发展" —— 太抽象，"有利"还要再定义
- 好判准：直接诠释"必须" —— "必须 = 缺它则儿童文学不成立"

## 四、战场切割（"剑宗"核心功夫）

### 1. 战场 = 讨论范围（黄执中定义）

> **战场是设定，不是扩大。**

辩手要做的不是把讨论扩大到所有相关内容，而是**框定**一个对自己有利的子集，
只在这个子集内交锋。

### 2. 经典战场切割案例

#### 案例 1：「子女不应孝顺父母」（马薇薇）

切割两个前提：
1. 对父母好 ≠ 孝顺，也可以是"以德报德"；
2. 人与人交往的原则是"以直报怨、以德报德"。

→ 战场被框成："要不要做'自己对父母的好大于父母对自己的好'的行为"

→ 在这个战场上，反方"不应孝顺"天然占优。

#### 案例 2：「官员应不应因生活作风问题受行政处分」（反方难辩）

战场切割：
- 把"生活作风"限定为**"不违法但有违道德的行为"**；
- 渣男 A 喜欢一夜情 vs 嫖娼 vs 强奸 —— 后两者是违法 / 犯罪，不属于"生活作风"。

→ 战场被框成："不违法的私德要不要由公权力来处分"

→ 反方瞬间占优：私德不该由公权力管。

#### 案例 3：「人格独立 vs 经济独立 必要条件」

战场切割：
- 把"必要条件"定义在**逻辑学严格意义**（无 A 则无 B）；
- 把"举证责任"分配到正方（必须证明全称命题）。

→ 战场被框成："存不存在'无经济独立而人格独立'的人"

→ 反方只需举一个反例即胜。

### 3. 战场切割三问

立论前必问：
1. **本场最有利于我方的战场是什么？**（不是辩题给定的战场，是我方挑出来的）
2. **怎样的两个前提能把对方逼到这个战场上？**
3. **如果对方拒绝接受这两个前提，我方还有退路吗？**

## 五、论点设计

### 1. 论点的"三层结构"（武汉大学传统）

每个论点应当：
- **个人层** —— 对一个人的影响
- **社会层** —— 对一群人 / 一个共同体的影响
- **时代层** —— 对一个时代 / 价值观的影响

三层论点形成"层层递进"的美感。

### 2. 论点的"阴阳两面"打法（知乎实战）

同一个论点可以：
- **阳面**：有他多好（正面论证）
- **阴面**：没他多不好（反面归谬）

例：「离婚率上升是否社会进步」反方
- 阳面：离婚率下降时社会多和谐；
- 阴面：离婚率上升时社会有多阴暗。

### 3. 论点的"高打 / 低打"选择

| 情境 | 打法 |
|---|---|
| **优势战场** | 高打 —— 升价值、谈意义、上格局 |
| **劣势战场** | 低打 —— 谈实际、说困难、拉回常识 |

例：「全民医保该不该推行」
- 正方优势 → 高打："这是大国担当"
- 反方劣势 → 低打："理想很丰满，现实是医保水平会被拉低"

### 4. 论点的"按环节输出"

❌ 烂操作：二辩环节把所有论点全打一遍
✅ 好操作：二辩主推一个论点，三辩再推一个

→ 论点在时间上分布，让评委能记住每一个，而不是混成一锅粥。

## 六、攻防表的设计（赛前必做）

### 1. 攻防表结构

| 序号 | 对方可能攻击 | 我方主回应 | 备用反击 | 方法论标签 | 风险等级 | 证据 URL |
|---|---|---|---|---|---|---|

### 2. 三种典型攻击 → 反击模板

| 对方打法 | 反击逻辑 | 话术 |
|---|---|---|
| **A 推不出 B** | 给出推论链中的具体一步证据 | "我方的推论是 A→C→B，您方忽略了 C 这一步" |
| **没有 A，通过 C 也可以推出 B** | 证明 C 路径不普遍 / 成本高 | "C 路径成立需要 X 个特殊条件，不具普遍性" |
| **B 不重要** | 升价值 | "B 不仅是重要，B 是 Y 价值的载体" |

### 3. "守点"与"攻点"区分（必须明确）

| 类型 | 特征 | 战术 |
|---|---|---|
| **攻点** | 逻辑短、易论证、利己 | 优先打、反复打、追问到底 |
| **守点** | 逻辑长、不易论证 | 不主动打，被攻才解释，解释完立刻转攻点 |

> ❌ **切忌把守点打成攻点** —— 事倍功半，反而暴露弱点。

## 七、一辩的责任清单

| 任务 | 评判 |
|---|---|
| 把定义钉死，让定义贯穿全场 | 评委能复述你的定义 |
| 把判准说清，让评委有尺子 | 评委记得你的判准 |
| 给出 3 个论点，每个带 1 个例子 + 1 个数据 | 评委能记住至少 2 个 |
| 控制节奏，3'30" 不要赶完 | 评委不觉得你紧张 |
| 留好质询应对（首质会被攻击） | R2 后我方架构没崩 |

> 一辩立论的好坏，决定了后场三个辩位的难易程度。**一辩稿值得反复打磨 5 遍以上。**
