# 逻辑谬误手册

> **使用指南**：
> - 质询时：识别对方哪类谬误，按"反击模板"打回去；
> - 评委点评时：用专业谬误术语解释胜方为何赢、败方为何输；
> - 立论时：自查己方论证是否中了任何一种 —— **自己先别犯**。

---

## 一、分散注意力类（5 种）

### 1. 稻草人谬误（Strawman）

**特征**：歪曲对方观点，再攻击你歪曲过的版本。

**示例**：
> 小明：国家应当向教育领域投入更多资金。
> 小亮：你居然想减少国防开支，让我们手无寸铁！

**识别**：对比对方原话和你方复述，看是否一致。

**反击模板**：
```
"请对方辩友复述一遍我方原话。我方说的是 X，不是您方理解的 Y。
请评委注意，对方在歪曲我方观点。"
```

---

### 2. 红鲱鱼 / 转移话题（Red Herring）

**特征**：用一个看似相关、其实无关的话题转移焦点。

**示例**：
> A：钱可以买到爱情，因为约会、生孩子都要钱。
> 实质：把"爱情可以用钱获得"偷换为"爱情可以用钱支撑"。

**反击模板**：
```
"对方在偷换话题。我方讨论的是 A，您方讨论的是 B，请回到 A。"
```

---

### 3. 复合问题（Complex Question）

**特征**：一个问题里塞两个或多个问题，让对方无论怎么答都中招。

**经典示例**："你停止打你老婆了吗？" —— 答 yes 等于承认打过，答 no 等于现在还在打。

**反击模板**：
```
"这其实是两个问题：1) 我是否曾经 X？2) 我现在是否还在 X？我分别回答..."
```

---

### 4. 诉诸权威（Appeal to Authority）

**特征**：用名人 / 权威说的话当论据，不讲背后的逻辑。

**示例**：爱因斯坦说过 X，所以 X 是对的。

**反击模板**：
```
"名人也会错。请对方给出 X 这个观点本身的逻辑，而不是引述谁说过。"
```

---

### 5. 诉诸民意（Argumentum ad Populum）

**特征**：很多人相信，所以是对的。

**示例**：80% 的人都同意 X，所以 X 是对的。

**反击模板**：
```
"多数人支持的不一定正确——历史上多数人也认为地球是平的。
请对方给出 X 本身的合理性论证。"
```

---

## 二、归因错误类（4 种）

### 6. 错误归因（False Cause / Post Hoc）

**特征**：因为 A 发生在 B 之前，就认为 A 导致 B。

**示例**：
> 我穿了红袜子，今天考试 100 分，所以红袜子是幸运色。

**反击模板**：
```
"A 和 B 只是先后发生，但不一定有因果关系。请证明 A → B 的具体机制。"
```

---

### 7. 滑坡谬误（Slippery Slope）

**特征**：把"可能性"夸大为"必然性"，一连串假设直接推出极端后果。

**示例**：
> 如果允许同性婚姻，将来就会有人和电脑结婚！

**注意**：滑坡谬误**不是**所有连锁推论都是谬误。如果每一步都有充分论据，
就是合理推论；缺论据时才是滑坡。

**反击模板**：
```
"对方做了一个滑坡推论，从 A 到极端结果之间有 B、C、D 三步。
请证明每一步必然发生，否则这只是恐吓。"
```

---

### 8. 以偏概全（Hasty Generalization）

**特征**：用少数案例推论普遍规律。

**示例**：
> 我认识一个北方人很凶，所以北方人都很凶。

**反击模板**：
```
"对方用了 N 个案例就得出全体结论。
N 个案例不构成统计学意义上的代表性，请评委注意。"
```

---

### 9. 关联混淆因果（Correlation ≠ Causation）

**特征**：A 和 B 相关，就说 A 导致 B。

**示例**：
> 冰淇淋销量和溺水率正相关，所以吃冰淇淋导致溺水。
> 实质：两者都因"夏天"而升高，是第三变量。

**反击模板**：
```
"相关不等于因果。X 和 Y 可能都受 Z 影响。请排除 Z 因素再下结论。"
```

---

## 三、对人不对事类（5 种）

### 10. 人身攻击（Ad Hominem）

**特征**：攻击对方个人特质，而非对方的论点。

**示例**：
> 你才大学生懂什么经济？

**反击模板**：
```
"我个人的身份与我的论点无关。请对方回到论点本身。"
```

---

### 11. 诉诸虚伪 / 你也一样（Tu Quoque）

**特征**：指出对方与其观点不一致的行为，以此反驳观点。

**示例**：
> A："抽烟有害健康。"
> B："你自己也抽烟，凭什么劝我！"
> 实质：B 没反驳"抽烟有害"，只攻击 A 个人。

**反击模板**：
```
"我个人是否做到，与命题本身是否正确，是两个不同的问题。
即使我没做到，也不影响命题为真。"
```

---

### 12. 扣帽子 / 井里下毒（Poisoning the Well）

**特征**：提前给对方贴标签，让听众预先排斥对方接下来要说的话。

**示例**：
> 在对方发言前说："接下来你们会听到一个典型的'极端自由派'的观点。"

**反击模板**：
```
"请评委注意，对方在我方发言前先扣帽子。我方观点与任何标签无关，请直接评判内容。"
```

---

### 13. 罪恶关联（Guilt by Association）

**特征**：因为 X 也支持某观点，所以这个观点是错的。

**示例**：
> 希特勒也支持素食主义，所以素食主义是错的。

**反击模板**：
```
"X 支持的不一定错，X 反对的也不一定对。请评判观点本身，不要看谁支持。"
```

---

### 14. 诉诸情感（Appeal to Emotion）

**特征**：用情感煽动代替逻辑论证。

**子类型**：
- 诉诸怜悯：博取同情
- 诉诸恐惧：制造恐慌
- 诉诸仇恨：激起厌恶

**反击模板**：
```
"感动 / 害怕 / 愤怒 都不等于正确。请回到事实和逻辑层面。"
```

---

## 四、虚假二分类（3 种）

### 15. 非黑即白 / 假二难（False Dilemma）

**特征**：给出 2 个极端选项，假装没有中间地带。

**示例**：
> 你要么支持我们，要么就是敌人。

**反击模板**：
```
"对方给的两个选项都不完整，实际上还有第三、第四种可能：X、Y。"
```

---

### 16. 完美主义谬误（Nirvana Fallacy）

**特征**：因为方案不完美，所以拒绝方案。

**示例**：
> 这个慈善项目还是有人没受益，所以是失败的。

**反击模板**：
```
"不完美不等于失败。请对方给出更好的替代方案，否则就是在用'完美'为'什么都不做'辩护。"
```

---

### 17. 中庸谬误（Argument to Moderation）

**特征**：认为折中观点总是正确的。

**示例**：
> 一方说 1+1=2，一方说 1+1=3，所以 1+1=2.5。

**反击模板**：
```
"折中不等于正确。在某些问题上，一方就是对的，另一方就是错的，没有中间地带。"
```

---

## 五、循环与举证类（4 种）

### 18. 循环论证（Circular Reasoning / Begging the Question）

**特征**：用结论本身当论据。

**示例**：
> 《圣经》是真的，因为《圣经》上说它是上帝写的。

**反击模板**：
```
"对方在用 A 证明 A。请给出 A 之外的独立证据。"
```

---

### 19. 诉诸无知（Argument from Ignorance）

**特征**：因为某事未被证明为假，所以是真的。

**示例**：
> 没有人能证明鬼不存在，所以鬼存在。

**反击模板**：
```
"未被证伪不等于已被证实。举证责任在主张存在的一方。"
```

---

### 20. 举证倒置（Burden Shifting）

**特征**：把举证责任推给反对者。

**示例**：
> A："X 是真的。"
> B："请证明。"
> A："你能证明 X 是假的吗？"

**反击模板**：
```
"举证责任在主张者一方。您方主张 X，请您方证明 X。
不是我方要证明 X 不存在。"
```

---

### 21. 自我打脸 / 自相矛盾（Self-Contradiction）

**特征**：对方前后两句话互相矛盾。

**示例**：
> "绝对没有绝对的事情。"

**反击模板**：
```
"对方在 R___ 说了 X，又在 R___ 说了非 X。请问您方真实立场是哪个？"
```

---

## 六、概念偷换类（3 种）

### 22. 偷换概念（Equivocation）

**特征**：在论证过程中悄悄改变一个词的含义。

**示例**：
> 自由是法律允许的；法律是束缚自由的；所以法律既允许又束缚自由。
> 实质：第一句"自由"指"权利"，第二句"自由"指"想做什么做什么"。

**反击模板**：
```
"对方在 R___ 使用 X 的含义是 A，在 R___ 使用 X 的含义变成了 B。
请确认您方对 X 的定义统一。"
```

---

### 23. 范畴错误（Category Error）

**特征**：把不同范畴的事物放在一起比较。

**示例**：
> "苹果比正义更重要" —— 实物与抽象概念无法比较。

**反击模板**：
```
"对方在比较两个不同范畴的事物。X 是 A 类，Y 是 B 类，无法直接比较。"
```

---

### 24. 不当类比（Faulty Analogy）

**特征**：用本质不同的事物做类比。

**示例**：
> 枪和铁锤都是金属工具，铁锤合法，所以枪也应该合法。
> 实质：枪和铁锤的核心用途、危害程度完全不同。

**反击模板**：
```
"X 和 Y 的核心特征不同。X 具有 A、B、C 特征，Y 具有 D、E、F 特征。
您方的类比忽略了关键差异。"
```

## 七、辩论场上的"高频中招"清单

按出现频率排序，新手辩论最容易中以下 5 种：

| 排名 | 谬误 | 中招情境 |
|---|---|---|
| 1 | 以偏概全 | 举一个例子就推论普遍 |
| 2 | 滑坡谬误 | 害怕反方反击，把己方论点推得太远 |
| 3 | 偷换概念 | 一辩稿和三辩稿用同一个词不同含义 |
| 4 | 诉诸情感 | 没论据时改用煽情 |
| 5 | 稻草人 | 反驳对方时其实在反驳自己理解的版本 |

→ **赛前自检**：把己方一辩稿过一遍，自查是否中了以上任何一项。

## 八、反击的"礼貌话术"（避免咄咄逼人）

| 直白话术 | 礼貌升级版 |
|---|---|
| "您在偷换概念！" | "请对方再次确认 X 这个词在两处的含义是否一致。" |
| "您在稻草人！" | "请对方先复述我方原话，再做反驳。" |
| "您滑坡了！" | "请对方为推论链中每一步给出论据。" |
| "您诉诸情感！" | "感动是真挚的，但我们还要回到事实层面来评判。" |
| "您以偏概全！" | "您方拿出的是个例，请提供统计层面的支撑。" |

→ 评委更喜欢**冷静指出**而非**情绪化指责**的辩手。

## 九、评委点评中如何用谬误术语

评委在点评时使用专业谬误术语，能立刻让点评显得**权威**。

### 模板：

```
"反方在 R7 三辩质询正方二辩时，使用了**归谬法** —— 把对方的论点
推到极端情境，让对方自己感到尴尬。这一招属于**逻辑层面的有效攻击**。

而正方二辩在回应时陷入了**举证倒置** —— 要求反方证明反例不存在，
但举证责任本就在正方。这一点裁判要扣分。"
```

→ 用了 3 个专业术语（归谬法、逻辑层面、举证倒置），点评立刻有质感。

## 十、自我检验：你的论证有没有中招？

赛前过一遍这 10 个问题，每个回答"否"才能上场：

- [ ] 1. 我的论点是否用我的结论本身作为论据？（循环论证）
- [ ] 2. 我是否用极端例子推全称结论？（以偏概全）
- [ ] 3. 我的推论链是否每一步都有独立论据？（滑坡）
- [ ] 4. 我使用的关键词在不同段落含义是否一致？（偷换概念）
- [ ] 5. 我引用的权威是否真的支持我的具体观点？（诉诸权威）
- [ ] 6. 我是否在用 A 相关 B 论证 A 导致 B？（混淆因果）
- [ ] 7. 我是否歪曲了对方的原话再攻击？（稻草人）
- [ ] 8. 我是否用情感代替了论证？（诉诸情感）
- [ ] 9. 我是否只给了非黑即白的两个选项？（假二难）
- [ ] 10. 我的类比是否抓住了本质特征？（不当类比）

→ **辩论中最难的不是攻别人，是自己不中招。**
