# 质询与攻防

## 一、质询的本质

> **质询的目的不是"驳倒对方"，而是"削弱对方在评委心中的可信度"。**
> 攻击的是对方在论证过程中的**逻辑推理**和**事实论据**。
> —— 知乎《零基础也能学质询：一看就懂的质询教学文》

质询手必须有两项能力：
1. **提炼对方论证线条的能力**（听 = 拆 = 复述）
2. **发现破绽的能力**（找 = 攻 = 推论）

---

## 二、质询四步法 · 起追结转（武大 / 新国立模板）

> 这是华语辩论最经典的质询节奏。

| 步骤 | 动作 | 时长 |
|---|---|---|
| **起** | 把问题的逻辑说清楚，然后提出第 1 个问题 | 15-30 秒 |
| **追** | 用 2-3 个**同样逻辑**的问题追下去 | 30-60 秒 |
| **结** | 把结论结一下，明确己方取得的战场成果 | 10-20 秒 |
| **转** | 转到下一组逻辑，另一个攻击点 | 5 秒过渡 |

**完整模板**（一场 2'30" 首质）：
```
起：对方一辩，您方刚才把 X 定义为 Y——这个定义里有一个前提是 Z 吗？
追 1：如果 Z 成立，那么按您方逻辑，A 也应该满足，对吗？
追 2：但现实中我们看到 B 案例，B 显然不满足 Z，您怎么解释？
追 3：所以您方的判准在 B 这种情况下其实是失效的，对吗？
结：（转向评委）请评委注意，对方在 B 类情境下无法解释。
转：好，我们看下一个问题——您方第二个论点说 ...
```

## 三、质询的"二段法" · 确认 + 处理（知乎完整版）

### 阶段 1：确认（≈30% 时间）

#### 1.1 框定（Framing）

把对方的论证用**你方有利的措辞**复述出来，让对方在你设定的框架内回答。

```
× 烂问法："您方说的是什么意思？"（开放，对方乱回答）
√ 好问法："您方刚才主张 X 是因为 Y，对吗？"（封闭，对方只能答是/否）
```

#### 1.2 确认共识（Yes-set）

用 2-3 个对方**必然会同意**的问题，让对方说"是"。
```
"我们都同意 A 吧？"
"我们也都承认 B 吧？"
"那 C 也是共识吧？"
```
连续 3 个 Yes 之后，对方的批判性思维就麻痹了，后面的杀招更容易成功。

### 阶段 2：处理（≈60% 时间）

#### 2.1 削弱可信度

攻击对方**论据**：
- 数据来源不权威 / 样本不具代表性
- 案例是个例不能代表普遍
- 引用断章取义

攻击对方**推理**：
- A 推不出 B（逻辑跳跃）
- 概念被偷换
- 举证倒置

例（旧版"成人网站合理性"辩论）：
> 对方论据："广东某高校调查显示，通过浏览成人刊物激发性幻想的男生比例最高 41.6%"
> 质询：
> Q1：这是广东某高校，不是全国数据，对吗？
> Q2：41.6% 是男生子集，不是全体大学生，对吗？
> Q3：这个数据只说"激发性幻想"，没说"对成人网站充分认同"，对吗？
> 结：所以您方用一个**地域局限 + 性别局限 + 概念偷换**的数据，论证不了"大众对成人网站充分认同"。

#### 2.2 激化观点

把对方观点**推到极端**，让评委自己看出荒谬。

```
"按您方逻辑，是不是只要 X，就一定 Y？"
"那如果出现 Z（极端情境），您方还坚持 Y 吗？"
```

### 阶段 3：展示（≈10% 时间）

转向评委做小结：
```
"请评委注意，对方在 X 这一点上 [回避 / 失答 / 让步]。"
"所以对方的论证在 [具体处] 已经站不住。"
```

## 四、质询的"防御技巧"（被质询方）

### 1. 防打断机制（黄执中招）

> 在回答的开头说："这个问题有**两个 / 三个**层面，我分别说——"
>
> 对方一旦打断你，会显得"不让你说完"，评委印象分扣他。

### 2. 不算对方时间（单边计时利用）

被质询方的发言**不计入**质询方的时间。
所以遇到难答的问题，可以**故意拉长回答**，让对方时间流失。

### 3. 不回答假设性问题

```
× 对方："如果 X 发生，您方怎么办？"
× 你："X 发生的话，我..."（中招了）
√ 你："我们先确认 X 在现实中是否成立。X 不成立的前提下，这个问题就是伪问题。"
```

### 4. 主动换例

如果对方拿一个你不熟悉的例子刁难，**直接说**："这个例子比较小众，我们换一个更典型的例子——"
评委不会觉得你怂，反而觉得你大方。

### 5. 不能反问质询方（规则红线）

**被质询方不允许反问。** 但可以"陈述质疑"：
```
× "那您方怎么解释 X？"（这是反问，违规）
√ "我方对您方的 X 主张持保留意见，因为..."（这是陈述）
```

## 五、首质（一辩 ↔ 四辩）的特殊任务

首质 = 第一轮质询（R2 反四质询正一 / R4 正四质询反一）。

除了通用质询规则外，**必须**完成三件事：

| 任务 | 怎么做 |
|---|---|
| **钉死对方定义** | "您方对 X 的定义边界在哪里？包括 / 不包括 Y？" |
| **钉死对方判准** | "您方判准是 X，那么按这个判准 Y 算赢还是输？" |
| **找出双方分歧** | "好，我们已经确认双方在 A 上同意，分歧在 B 上对吗？" |

→ 把分歧钉死，留给后场使用。

## 六、三辩质询的"短而狠"特点（R7、R9）

三辩质询只有 1'30"，时间紧张，必须：

- **聚焦 1 个攻击点**（不要贪多）
- **4 个回合内**打出明确攻击
- **拿下即小结**（必做）

模板：
```
Q1：您方二辩刚才主张 X，这里有个前提是 Y，对吗？（确认）
Q2：那么按 Y 这个前提，A 案例是不是反例？（追问）
Q3：A 案例是真实存在的，您方怎么解释？（逼问）
Q4：（如果回避）请评委注意，对方回避了 A 案例。所以 Y 这个前提其实不普遍。（小结）
```

## 七、自证 / 打岔的攻防（R10、R11）

### 1. 自证方的核心任务

| 任务 | 做法 |
|---|---|
| 推进 1 条主轴论点 | 选场上最强、最未被反驳的论点深耕 |
| 回应前场关键战场 | ≥2 处实质回应，不能完全无视前场 |
| 战略性忽略某次打岔 | 不必每次都回应，节奏在自己手里 |

### 2. 打岔方的"狠"技巧

- 打岔 ≤ 3 次，每次 ≤ 20 秒（≤ 95 字）；
- **每次打岔必须带 1 个具体例子**或 **1 个谬误指出**；
- **不能只是"我反对"** —— 没用，浪费时间；
- 高级打法：用 1 个排比把自证方的推论瞬间归谬。

例：
> 自证方："经济独立给我们带来选择的自由..."
> 打岔："那么婴儿没有自由？学生没有自由？被赡养的老人没有自由？您方的'自由'定义太狭窄了。"（一个排比 + 一个反问，20 秒结束）

## 八、自由辩论的攻防细节

### 1. 起追结转的快速版

自由辩论每次发言只有 30-45 秒，必须把"起追结转"压缩到 15 秒内完成：
```
"对方刚才说 X，请问 Y 情况下是不是反例？...所以您方在 Y 上其实站不住。
我方下一个问题——..."
```

### 2. 自由辩论的中段提速短打（马来亚大学模板）

中段把每个问题压缩到 10 秒左右，揪住对方难解释的点**连续追问**：
```
"X 怎么解释？"（10 秒）
对方答 30 秒（不算我方时间）
"还是没回答，X 怎么解释？"（5 秒）
对方再答 30 秒
"第三次问 X 怎么解释？请评委注意对方回避了 3 次。"（10 秒）
```

→ 节奏感强，给评委留下"对方在回避"的印象。

### 3. 王牌辩手盯防（北大模板）

派己方最强辩手**专门盯防**对方王牌辩手，每次他发言完，立刻接他的话刚正面。
- 让对方王牌没有发挥空间；
- 制造心理压制。

## 九、质询/自由辩的常见错误

| 错误 | 表现 | 后果 |
|---|---|---|
| **只问不推** | "是不是？""对不对？"连珠炮，但没结论 | 评委记不住你打了什么 |
| **多线逼近** | 一会儿打 A 一会儿打 B，跳来跳去 | 哪条都没打透，全是浅尝辄止 |
| **重复上轮** | 反复问同一个对方已答的问题 | 显得没新东西，被裁判扣分 |
| **答非所问** | 被质询时绕开问题 | 评委看得出来你在回避 |
| **抢答** | 对方还没说完就打断（除规则允许打断的环节） | 礼貌分扣，评委印象差 |
| **小结失败** | 拿下了观点却没小结 | 拿下了也白拿，评委不记得 |

## 十、质询的"心法"

> **辩论中最神奇的地方，是如何用应然改变实然。** —— 路一鸣

实然层面：你打不动对方某个事实（比如某个数据是真的）。
应然层面：你可以质疑"这个数据是否应该被这样使用"。

质询时不要总想着"对方说的是不是真的"，而要想：
- **对方的论证模式是不是合理？**
- **对方的判准是不是公允？**
- **对方的归因是不是真有因果？**

→ 高阶质询打的是**论证方法论**，不是**事实层面**。
