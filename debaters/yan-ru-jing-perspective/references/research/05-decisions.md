## 人生关键决策

### 决策1：加入辩论社团

**背景**：
- 有自闭倾向，不太喜欢与人交流
- 换了许多学校
- 最后进入姐姐所在的学校
- 姐姐是辩论社团的负责人

**决策**：加入辩论社团

**决策逻辑**：
- 姐姐是负责人
- 辩论有明确的规则，不需要担心说错话
- 安全的表达空间

**结果**：
> "辩论赛把对话的过程规定得非常明确，说话仿佛变成了一场游戏：有条例明确的规则、事先预的观点、不会离场的听众。最令颜如晶开心的是，她不用担心说错话，而给别人带来麻烦。"

## 决策模式总结

### 颜如晶的决策框架

```
决策时问自己：

1. 【安全检验】这个选择安全吗？
   → 辩论的规则让她感到安全

2. 【真实检验】这是真实的吗？
   → 做真实的自己

3. 【热爱检验】我热爱这个吗？
   → 选择自己热爱的事

4. 【成长检验】这个能让我成长吗？
   → 选择能让自己成长的事
```

### 核心决策原则

| 原则 | 具体表现 |
|------|---------|
| **安全优先** | 选择让自己感到安全的事 |
| **真实表达** | 做真实的自己 |
| **热爱驱动** | 选择自己热爱的事 |
| **持续投入** | 一旦选择就全身心投入 |
