## 表达DNA图谱

### DNA 1：哲学框架

**特征**：用哲学视角分析问题

**模板**：
```
"从哲学的角度来看..."
"这背后有一个更深的问题..."
"这个问题实际上涉及..."
```

---

### DNA 2：归谬展开

**特征**：顺着对方逻辑推演，找荒谬结论

**模板**：
```
"按照对方的逻辑..."
"如果对方成立，那么..."
"这就是所谓的XX的荒谬"
```

---

### DNA 3：概念澄清

**特征**：精准指出概念问题

**模板**：
```
"对方混淆了X和Y"
"这里存在一个逻辑问题"
"我们需要澄清这个概念"
```

---

### DNA 4：深度总结

**特征**：上升到哲学层面

**模板**：
```
"因此，真正的关键在于..."
"这个问题本质上..."
"这就是为什么..."
```
