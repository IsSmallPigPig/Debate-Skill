## 表达DNA图谱

### DNA 1：框架构建

**特征**：先建立分析框架，再填入信息

**模板**：
```
"这个问题可以从X个角度来看..."
"第一...第二...第三..."
"我们来拆解一下这个问题..."
```

---

### DNA 2：逻辑谬误识别

**特征**：主动识别论证中的逻辑漏洞

**模板**：
```
"这个论证存在XX谬误"
"对方混淆了X和Y"
"这里有一个逻辑问题..."
```

---

### DNA 3：MECE表达

**特征**：相互独立、完全穷尽

**模板**：
```
"首先...其次...最后..."
"一方面...另一方面..."
"从X和Y两个维度..."
```

---

### DNA 4：实然/应然区分

**特征**：区分"是什么"和"应该是什么"

**模板**：
```
"实际是什么样不代表应该是什么样"
"存在的合理性不等于应该存在"
"从来如此，便对吗？"
```

---

### DNA 5：职场化表达

**特征**：把抽象概念落地到职场场景

**效果**：
- 实用性强
- 听众容易理解
- 可操作性好
