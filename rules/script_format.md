# Script Format · 真实辩稿格式规范

> **核心原则**：辩稿是"辩手要逐字念出来的稿子"，不是"导演剧本"。
> 写出来的东西必须能**直接打印**给辩手在场上朗读。

---

## 1. 通用红线（任何辩稿都不能违反）

### 1.1 严禁出现的内容

| ❌ 禁止 | 原因 | 反例 → 正例 |
|---|---|---|
| **动作括号** | 真实辩稿不写表演指令 | ❌ "（沉稳起立，目光扫向反方）"<br>✅ 直接进入"谢谢主席，各位评委…" |
| **表情/情绪括号** | 同上 | ❌ "（眉头紧锁）" "（拍桌）" "（深呼吸）"<br>✅ 删除即可 |
| **舞台说明** | 不是话剧本 | ❌ "（转向评委）" "（语速加快）"<br>✅ 删除 |
| **元说明** | 不是工作日志 | ❌ "我现在开始我的一辩稿："<br>✅ 直接念"谢谢主席…" |
| **AI 痕迹** | 破坏沉浸感 | ❌ "作为正方一辩，我将…"<br>✅ "我方坚定认为…" |
| **方括号占位符未替换** | 草稿味 | ❌ "[此处插入论点]" "{数据 TBD}"<br>✅ 必须填上具体内容 |
| **小说式描写** | 同上 | ❌ "马薇薇站起，犀利地扫视全场"<br>✅ 删除 |

### 1.2 必须出现的要素

| ✅ 必有 | 用法 |
|---|---|
| **问候语** | 开头第一句："谢谢主席，各位评委、对方辩友、各位同学，大家好。" |
| **立场重申** | 开头 30 秒内必须明确说出"我方认为/我方坚持/我方观点是…" |
| **结尾收束** | 必须用"综上所述…""我方坚定认为…""谢谢！"等真实辩场套语 |
| **段落标号** | 用"第一/第二/第三"或"首先/其次/再次/最后" |

## 2. 一辩稿（立论稿）格式

### 2.1 章节结构（必须按顺序）

```text
1. 问候语        （15–25 字，1 行）
2. 开篇立场      （30 秒，约 140 字，1 段）
3. 开宗明义·定义 （30 秒，约 140 字，1 段）
4. 判准          （30 秒，约 140 字，1 段）
5. 论点一        （45 秒，约 210 字，1 段，含证据）
6. 论点二        （45 秒，约 210 字，1 段，含证据）
7. 论点三        （45 秒，约 210 字，1 段，含证据）
8. 综上所述      （15 秒，约 70 字，1 段）
9. 谢谢
```

**总时长 3'30"，总字数 ≥ 980 字**（按 280 字/分钟）。

### 2.2 标准套语

| 章节 | 推荐开头句 |
|---|---|
| 问候语 | 谢谢主席，各位评委、对方辩友、各位同学，大家好。 |
| 开篇立场 | 今天我方的立场是 ___。 / 我方坚定认为 ___。 |
| 定义 | 开宗明义，___ 指的是 ___；___ 指的是 ___。 |
| 判准 | 基于此，我方判定 ___ 的标准是 ___。 |
| 论点一 | **第一，___**。理由在于 ___。例如 ___（数据/案例 + 出处）。 |
| 论点二 | **第二，___**。 |
| 论点三 | **第三，___**。 |
| 收束 | 综上所述，我方坚持认为 ___。谢谢。 |

### 2.3 完整范例（"行动比呐喊更重要"反方一辩稿节选）

```markdown
# 反方一辩稿｜行动比呐喊更重要

谢谢主席，各位评委、对方辩友、各位同学，大家好。

今天我方的立场是：**行动比呐喊更重要**。

开宗明义。呐喊，指为某项事物公开发声、倡议；行动，指脚踏实地去完成某件事。
我方判准是：在面对一个具体问题时，**哪一种方式更能推动问题被解决**，
哪一种就更重要。基于这一判准，我方将从以下三个层面展开论证。

**第一，从个人层面看，行动是呐喊的前提，没有行动支撑的呐喊只是空话。**
孔子曰："吾尝终日不食，终夜不寝，以思，无益，不如学也。" 表达不等于成事。
布鲁诺为日心说呐喊一生而被烧死，是牛顿用《自然哲学的数学原理》落地了
科学革命，世界才真正承认地动说。这说明：行动可以独立成事，呐喊不能。

**第二，从国家层面看，凡推动历史前进的关头，靠的都不是口号而是落地。**
1978 年中国改革开放，邓小平给出了"实践是检验真理的唯一标准"
这一总纲；接下来 1980 年深圳特区、1984 年城市改革、1992 年南巡，
都是一项项具体行动累积出今天的中国经济奇迹。反观当时只喊"两个凡是"
的人，没有落到行动，今天没人记得他们说过什么。

**第三，从社会层面看，行动本身就是最有感召力的呐喊。**
蚂蚁森林公益项目，至今约 5 亿用户、累计种下逾 1 亿棵真树。
这群人没有上联合国发言，他们只是默默积攒"绿色能量"，
然而正是这一行动本身，比任何一场演讲都更有力地向社会发出了
"我们在保护地球"的信号。

综上所述，呐喊固然有其价值，但**只有行动能让价值真正发生**。
我方坚持认为：行动比呐喊更重要。谢谢。
```

> 注意：**通篇没有一个动作括号、没有一个语气提示**。所有内容都是辩手要念出来的话。

## 3. 二辩稿 / 三辩稿（驳论 + 推进）格式

### 3.1 章节结构

```text
1. 问候语
2. 回应对方上一轮的具体攻击（≥1 处明确点名，约 200 字）
3. 推进本方架构（1–2 个新论点 或 深化既有论点，约 350 字）
4. 收束 + 谢谢
```

**总时长 2'30"，总字数 ≥ 700 字**。

### 3.2 标准套语

| 章节 | 推荐开头句 |
|---|---|
| 回应对方 | 在我开始之前，我必须先回应对方一辩刚才提到的 ___。对方说 ___，但事实上 ___。 |
| 推进 | 接下来，我要在我方一辩论点的基础上进一步指出：___。 |
| 收束 | 综上，我方立场不变：___。谢谢。 |

### 3.3 严禁

- ❌ 不准重复一辩已经说过的论点（违者本轮失败）
- ❌ 不准在二辩稿里现场修改一辩定下的定义/判准

## 4. 四辩稿（结辩稿）格式

### 4.1 章节结构（必须按顺序）

```text
1. 问候语                          （1 行）
2. 战场梳理                        （约 280 字，逐一标注 拿下/被拿/僵持 + 理由）
3. 攻防关键瞬间                    （约 200 字，引用 ≥2 处具体轮次）
4. 价值升华                        （约 200 字，呼应本方判准）
5. （正四专属）正面回应对方四辩    （约 150 字，仅正四）
6. （正四专属）给评委的判决建议    （约 100 字，仅正四）
7. 结尾金句 + 谢谢                 （约 50 字）
```

**总时长 3'30"，总字数 ≥ 980 字**。

### 4.2 标准套语

| 章节 | 推荐开头句 |
|---|---|
| 战场梳理 | 经过 12 轮的交锋，今天场上共有 N 个核心战场。**第一个战场是 ___**，对方在 R___ 主张 ___，我方在 R___ 用 ___ 回应，**这个战场我方拿下**，因为 ___。 |
| 攻防关键瞬间 | 我必须特别提两处。第一，R___ 反三质询正二时 ___；第二，R___ 我方自证 ___。 |
| 价值升华 | 回到我方一辩在 R1 / R3 定下的判准：___。今天我方所有努力都在守护 ___。 |
| 正四回应反四 | 反方四辩刚才说我方在 ___ 战场被拿下，但请评委回看 R___：___。 |
| 判决建议 | 如果各位评委接受 ___ 判准，我方理应胜出；如果接受 ___ 判准，请也注意 ___。 |
| 收束 | ___（金句）。谢谢！ |

### 4.3 严禁

- ❌ 不准现场新增定义
- ❌ 不准只复述，必须有判断（"我方拿下/被拿/僵持"必须明确说出来）

## 5. 质询稿（含三辩质询、四辩首质）格式

### 5.1 写法

质询稿不是一段连续散文，而是**预先设计好的问答脚本**。

```text
# 质询目标
（一句话写明这场质询要拿下对方什么具体观点）

# 主线问答

**Q1（背景设置）**: ___
- 预期答案 A: ___ → 我方走法：___
- 预期答案 B: ___ → 我方走法：___

**Q2（追问/收口）**: ___
- 预期答案 A: ___ → ___
- 预期答案 B: ___ → ___

**Q3**: ___
**Q4**: ___（拿下后立即小结）

# 风险预案
- 对方若答 ___ → 走支线 ___
- 对方若拒答 → 直接小结："请评委注意，对方回避了 ___ 这一问题。"

# 小结模板
"所以对方在 ___ 这一点上 站不住 / 与本方判准冲突 / 无法回应 ___。"
```

### 5.2 现场质询发言的写法

实际质询轮的对话，按问答交替写。**所有发言都是辩手要直接说的话**，
不带动作、不带情绪。

```text
**反方四辩**：对方一辩，您方将"幸福"定义为"主观满足感"。请问，
一个被洗脑后觉得自己很幸福的人，按您方定义，算幸福吗？

**正方一辩**：我方的定义并不排除认知层面的判断，所谓主观满足感
是建立在对客观处境的真实认知之上的。

**反方四辩**：所以您方承认，纯主观满足并不足以构成幸福，还需要
客观条件的支撑。是这样吗？

**正方一辩**：是这个意思。

**反方四辩**：好，请评委注意：对方已经承认幸福需要客观条件支撑。
那么我接下来问第二个问题——……
```

**严禁**：写成 "（打断）" "（语速加快）" 这种小说式标记。
**允许**：直接写发言内容；如需表达打断关系，**靠对话顺序**和**未完成的句号**自然呈现。

## 6. 备赛资料 / 前场整理 / 攻防表（赛中文档）格式

这些是**辩手私下用的资料**，不是辩场发言稿，可以用普通工作文档格式。
但同样不允许出现"（动作）"等小说式标记。

### 6.1 备赛资料 `{side}-备赛资料.md`

```markdown
# {正/反方} 备赛资料｜{辩题}

## 一、论点架构

### 1. 定义
- 关键词 1：___
- 关键词 2：___

### 2. 判准
___

### 3. 推论链
（从定义+判准 → 一步步推到本方观点，每一步要可被反驳）

### 4. 价值导向
___

## 二、资料收集

| # | 我方论点 | 资料标题 | 关键摘录 | URL | 来源类别 | 验证状态 |
|---|---|---|---|---|---|---|
| 1 | … | … | … | https://… | 学术/新闻/数据库 | 已二次验证 |

## 三、攻防表（赛前预演版）

| # | 对方可能的攻击 | 我方主回应 | 备用反击 | 方法论标签 | 风险等级 | 关键证据 URL |
|---|---|---|---|---|---|---|
```

### 6.2 前场整理 `{side}-前场整理.md`

```markdown
# {正/反方} 前场整理｜{辩题}

## 一、智识诚实：我方承认对方说得对的点
（≥1 个，写明哪一点、为什么承认）

## 二、前 4 轮战场回顾

| 战场 | 我方立场 | 对方立场 | 当前状态 | 关键轮次 |
|---|---|---|---|---|

## 三、后场执行计划
- **二辩 R6/R8**：要讲的新论点 / 要回应的攻击
- **三辩 R7/R9**：质询谁、目标拿下哪个点
- **三辩自证 R10/R11**：要推进哪条论点
- **四辩 R13/R14 结辩**：主轴一句话

## 四、经过前场后更锋利的最终立场
（一句话）
```

## 7. Markdown 排版规范（保证可直接复制到 Word 打印）

### 7.1 必须遵守

| 规则 | 说明 |
|---|---|
| **使用标准 Markdown** | `#` 标题、`**粗体**`、`- 列表`、`\| 表格 \|` |
| **中英标点正确** | 中文段落用全角标点（，。！？："""'') |
| **段落之间空一行** | 复制到 Word 后能保持段落分隔 |
| **不使用 HTML 标签** | 不写 `<br>`、`<p>`、`<div>` 等 |
| **不使用代码块嵌套辩稿正文** | 辩稿正文是普通段落，**不**包在 ` ``` ` 里 |
| **数字与单位分开** | "约 1.2 亿人" 而不是 "约1.2亿人" |
| **引用必须带 URL** | 写在段落末尾的（来源：https://...）或脚注 |

### 7.2 文件输出格式约定

| 文件类型 | 标题层级 | 是否分节 |
|---|---|---|
| 辩稿（一辩/二辩/三辩自证/四辩结辩） | `# {辩位}稿｜{辩题}` | 章节用 `##` |
| 质询稿 | `# {X质Y}质询稿｜{辩题}` | 主问 `##`、子问 `###` |
| 备赛资料 | `# {正/反方}备赛资料｜{辩题}` | `## 一/二/三` |
| 攻防表 | `# {正/反方}攻防表｜{辩题}` | 主体是表格 |
| 辩论实录 | `# 辩论实录｜{辩题}` | 每轮 `## 第 N 轮 · {环节}` |

### 7.3 可直接 Word 打印的最小要求

打印前自检：

- [ ] 无任何"（动作/表情/语气）"括号
- [ ] 无 `{占位符}` 未替换
- [ ] 无 "TODO" / "TBD" / "待补充"
- [ ] 段落之间有空行
- [ ] 表格列宽一致（` | 列1 | 列2 | `）
- [ ] 中文标点统一使用全角
- [ ] 标题层级不跳级（`#` 后不直接接 `###`）
- [ ] 引用资料带 URL
- [ ] 通篇没有 "作为 AI" / "我会扮演" 等元说明

## 8. 现场对话呈现格式（区别于"辩稿文件"）

> 现场辩论的对话呈现 ≠ 辩稿文件。
> 对话呈现要让用户**像在看比赛直播**，但同样**禁止小说式标记**。

### 8.1 标题行

每一轮用三级标题标注：

```markdown
## 第 1 轮 · 申论｜正方一辩 · 胡渐彪
> 时长 3'30" · 计时方式：—

谢谢主席，各位评委、对方辩友……
（完整辩稿正文，不带任何动作括号）

【完成】轮次=R1 时长=3'30" 字数=985 含3论点=是 含定义判准=是 含证据=3
```

### 8.2 质询/对话轮

用**辩位名加粗 + 冒号**呈现，不用动作括号：

```markdown
## 第 2 轮 · 质询｜反方四辩 → 正方一辩
> 时长 2'30" · 单边计时

**反方四辩 · 马薇薇**：对方一辩，您方将"幸福"定义为"主观满足感"。
请问，一个被洗脑后觉得自己很幸福的人，按您方定义算幸福吗？

**正方一辩 · 胡渐彪**：我方的定义并不排除认知层面的判断。

**反方四辩 · 马薇薇**：那么请正面回答：算还是不算？

**正方一辩 · 胡渐彪**：在主观满足这一指标上算，但是否构成
完整意义上的"幸福"还要看判准。

**反方四辩 · 马薇薇**：好，请评委注意，对方已经承认 ___。
那么我问第二个问题……

（继续 ≥ 5 个完整 Q-A 回合）

**反方四辩（小结）**：所以对方的定义在"被操纵的幸福"这一点上
是站不住的。

【完成】轮次=R2 时长=2'30"单边 总字数=720 完整回合=5 已钉定义=是 已做推论=是
```

### 8.3 严禁出现

❌ "（打断）" "（沉吟两秒）" "（拍桌）" "（转向评委）" "（深呼吸）"
❌ "马薇薇站起" "胡渐彪苦笑" 等小说式描写
❌ "现场气氛紧张" "全场鸦雀无声" 等旁白
❌ 出现以上任一项 → **本轮失败，必须重发 prompt 要求删除全部括号和旁白**

## 9. 自检清单（每份产出都要过）

辩手子代理或编排器在输出任何辩稿/对话之前，**默念**：

```text
[ ] 我有没有在文中加（动作/表情/语气）括号？      有 → 删掉
[ ] 我有没有写"马薇薇站起""胡渐彪沉吟"等旁白？  有 → 删掉
[ ] 我有没有用"作为正方一辩"这种元说明开头？     有 → 改成"谢谢主席…"
[ ] 我有没有把 {占位符} 替换成具体内容？        没 → 补上
[ ] 我有没有用真实辩场套语（开宗明义/综上所述）? 没 → 加上
[ ] 我的段落之间有没有空行？                    没 → 加上
[ ] 我引用的数据有没有带 URL？                  没 → 补上
[ ] 我的字数有没有达到时长换算下限？            否 → 重写
[ ] 我最后有没有附【完成】契约行？              没 → 加上
```

任何一项是"否" → **本轮不算完成**，必须修正后重发。
