# 辩论高手养成

> 从底层认知到临场实战，系统掌握辩论的六大维度 🎯

## 核心定位

辩论不是吵架，而是有逻辑、有结构、有策略的思想交锋。从底层认知、内容架构、攻防技术、语言表达、临场应变、心态团队六大维度，全覆盖辩论能力，新手到高阶都能直接套用。

## 六大核心维度

| 维度 | 核心内容 |
|------|---------|
| **底层认知** 🧠 | 逻辑思维、认知格局、知识储备 |
| **内容架构** 🏗️ | 定义标准、论点设计、论据运用 |
| **攻防技术** ⚔️ | 质询盘问、反驳拆解、自由辩交锋、结辩升华 |
| **语言表达** 🎭 | 语言功底、共情叙事、舞台呈现 |
| **临场应变** 💪 | 临场反应、节奏控场 |
| **心态团队** 🤝 | 心态抗压、团队协作 |

## 高手三层级

| 层级 | 能力 |
|------|------|
| **入门层** | 会立论、会反驳、有素材、说话清晰 |
| **进阶层** | 逻辑闭环、攻防熟练、能控场、会价值拉扯 |
| **高手层** | 定义博弈、视角降维、共情拉满、心态稳定、全场统筹 |

## 核心技能

### 逻辑模型
- 因果、并列、递进、类比、两难、归谬

### 攻防技术
- 质询盘问、反驳拆解、自由辩交锋、结辩升华

### 语言表达
- 精准、简洁、质感、共情、舞台呈现

## 核心理念

> **辩论是价值选择的艺术，不是对错的争辩。**

**输出**：

5. **攻防兼备** - 能攻善守
6. **心态稳定** - 逆风不慌
